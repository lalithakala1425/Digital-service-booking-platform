# ServEase – Digital Service Booking System

## 📌 Project Overview
ServEase is a frontend-based digital service booking platform that allows users to browse, select, and book services such as cleaning, beauty, repairs, and tutoring.

---

## 🚀 Features
- Browse different service categories
- View service details
- Filter services based on category and rating
- Book services with date and time
- Dashboard to view bookings
- Responsive modern UI
- Smooth animations

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom + Tailwind CSS)
- JavaScript (Vanilla JS)
- Local Storage (for saving bookings)
